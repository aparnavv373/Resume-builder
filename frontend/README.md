# 📄 AI-Powered Resume Builder



An intelligent resume builder application that uses AI to generate professional resume content, helping job seekers create impressive resumes quickly and effectively.

## ✨ Features

- **AI-Powered Content Generation**: Automatically generate professional descriptions for experience, projects, and summaries
- **Interactive Form Builder**: Step-by-step guided form for entering resume information
- **Responsive Design**: Mobile-first design with smooth navigation on all devices
- **Local Storage**: Automatic saving of your progress
- **Print-Friendly Preview**: Generate PDF-ready resume with professional formatting
- **Skill Management**: Add, remove, and organize skills with visual feedback
- **Real-time Validation**: Input validation with helpful error messages

## 🚀 Tech Stack

### Frontend
- **React 18** with Hooks
- **React Router DOM** for navigation
- **Tailwind CSS** for styling
- **React DatePicker** for date selection
- **React Icons** for beautiful icons

### Backend
- **FastAPI** (Python) for REST API
- **Groq API** with LLaMA 3.3 70B model for AI generation
- **CORS** middleware for cross-origin requests
