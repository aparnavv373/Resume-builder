# 📄 AI Resume Builder

A modern, feature-rich resume builder application that leverages AI to help users create professional resumes with ease. Built with React and FastAPI, this application provides an intuitive interface for building resumes with AI-powered content generation.

## ✨ Features

### Core Features
- **Interactive Form Builder**: Step-by-step guided process for entering resume information
- **AI-Powered Content Generation**: Automatically generate professional descriptions for experience, projects, and summary
- **Real-time Preview**: See your resume as you build it
- **PDF Export**: Download your resume as a professional PDF
- **Print-Friendly**: Print-ready format with proper page breaks
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Data Persistence**: Auto-saves your progress locally

### Sections
1. **Personal Information**: Contact details, location, and professional links
2. **Education**: Multiple education entries with degree and duration
3. **Experience**: Work history with AI-generated descriptions
4. **Projects**: Project details with AI-generated descriptions
5. **Skills**: Add and manage technical skills
6. **Professional Summary**: AI-generated summary based on your profile

## 🚀 Tech Stack

### Frontend
- **React 18** - UI Framework
- **React Router DOM** - Navigation
- **Tailwind CSS** - Styling
- **React Icons** - Icons
- **React DatePicker** - Date selection
- **html2pdf.js** - PDF generation

### Backend
- **FastAPI** - Python web framework
- **Groq API** - AI text generation
- **Python-dotenv** - Environment variables
- **Uvicorn** - ASGI server

## 📋 Prerequisites

- Node.js (v14 or higher)
- Python (v3.8 or higher)
- Groq API Key (for AI features)
- npm or yarn package manager
